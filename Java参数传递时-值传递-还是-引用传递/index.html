Java中对象的存储

堆：存储全部的对象（所有线程共享）
栈：存储基础数据类型和对象的引用

因为是值传递 所以p是person的一份拷贝，但是由于java的内存模型中，栈中存储的是对象的引用，
所以person是Person对象的引用，p拷贝拷贝的person也是Person的引用,所以运行change()函数之后，年龄会改变。
