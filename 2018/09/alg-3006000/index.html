有向图的相关算法小结

本文主要总结一些基础的有向图的算法，包括：有向图中的可达性，寻找有向环，拓扑排序，强连通分量

数据类型

有向图中的可达性

单点可达性：给定一幅有向图和一个起点s，回答“是否存在一条从s到达给定顶点v的有向路径？”等类似的问题。
使用深度优先搜索的方法，在图中进行从s进行深度优先遍历，然后查询点v是否被访问。

寻找有向环

判断有向图中是否存在环，是解决一些问题的基础，如优先级限制下的调度问题，拓扑排序等
解决深度优先搜索的方法解决有向环的思路：深度优先搜索过程中，递归调用的栈表示的正是“当前”正在遍历的有向路径，一旦我们找到一条有向边（v,w）且w已经存在于栈中，就找到了一个环，因为栈表示的是一条由w到v的有向路径，而（v，w）正好补全了这个环。

拓扑排序

给定一幅有向图，将所有的顶点排序，使得所有的有向边均从排在前面的元素指向排在后面的元素（或者说明无法做到这一点，即有环）

逆后续排列方法

如果将深度优先搜索的参数顶点保存在一个数据结构中，然后遍历这个数据结构就能访问每个顶点，遍历的顺序取决于这个数据结构的性质以及实在递归之前还是在之后进行保存数据，通常有以下三种排列顺序：

  1. 前序：在递归调用之前将顶点加入队列，前序就是dfs()的调用顺序
  2. 后续：在递归调用之后将顶点加入队列，后序就是顶点遍历完成的顺序
  3. 逆后续：在递归调用之后将顶点压入栈，逆后续就有向无环图的拓扑排序顺序

命题：一幅有向无环图的拓扑顺序就是所有顶点的逆后续排列

证明：对于任意一条边（v,w）在调用dfs(v)时，下面三种情况必有一种成立，
1）dfs(w)已经被调用过，且已经返回了（w已经被标记）
2）dfs(w)还没有被调用（w还未被标记），因此（v,w）会直接或间接调用并返回dfs(w),且dfs(w)会在dfs(v)返回之前返回
3）dfs(w)已经被调用但还未返回。证明的关键在于，在有向无环图中这种情况是不可能出现的，这是由于递归调用链意味着存在从w到v的路径，但存在（v，w）则表示存在环
而在1）和2）这两种情况中，dfs(w)都会在dfs(v)之前完成，因此在后续排列中w排在v之前而在逆后续排列中w排在v之后，因此任意一条边(v,w)都如我们所愿地从排名较前的顶点指向排名之后的顶点。

减治法实现

实现原理：不断地做这样一件事，在余下的有向图中求取一个源（source）（PS：定义入度为0的顶点为有向图的源），它是一个没有输入边的顶点，然后把它和所有从它出发的边都删除。（如果有多个这样的源，可以任意选择一个。如果这样的源不存在，算法停止，此时该问题无解）。

首先我们需要在有向图的数据结构Digraph中，增加一个数组用于保存所有的顶点的入度，同时在增加一条边的时候，顶点的入度加一。

强连通分量

若有向图G中的任意两个顶点vi和vj都是连通的，则称图G是强连通图。有向图G中的极大强连通子图称为G的强联通分量。

Kosaraju算法

第一步：计算出有向图G的反向图GR的逆后序排列
第二步：在有向图G中进行标准的深度优先搜索，按照刚才计算出的逆后序排列顺序而非标准顺序

Tarjan算法

tarjan基于DFS，用dfs遍历图G的所有点。分别建立两个数组Dfn和Low，Dfn[i]代表dfs到达顶点i的时间,Low[i]代表能够直接或者间接到达的最小顶点时间。
1.初始化，当首次搜索到点i时，Dfn与Low数组的值都为到该点的时间，Low[i]=Dfn[i]=time++。
2.将搜索到的点压入栈顶。
3.当点u与点v相连，如果v并不在栈中，先将v压入栈，dfs遍历v，则Low[u]=min(Low[u],Low[v]).
4.当点u与点v相连，如果v在栈中，则Low[u]=min(low[u],dfn[v]).
5.每当搜索到的点经过以上操作后，当Low[u]=Dfn[u]，则此时栈里u以及u以上的顶点全部出栈，即为一个极大强连通分量。
6.继续搜索，或许会更换搜索的起点，因为整个有向图可能分为多个不连通的部分，直到所有点被遍历。

private boolean[] marked;//已访问过的顶点
private int[] low;
private int[] dfn;//强连通分量的标识符
private int count;//强连通分量的数量
private Stack<Integer> stack;
private boolean[] inStack;
private int time;

private List<ArrayList<Integer>> result;

public void tarjan(Digraph G) {
    marked = new boolean[G.V()];
    low = new int[G.V()];
    dfn = new int[G.V()];
    stack = new Stack<>();
    inStack = new boolean[G.V()];
    result = new ArrayList<ArrayList<Integer>>();
    for (int s = 0; s < G.V(); s++) {
        if (!marked[s]) {
            dfs(G, s);
        }
    }
}

public void dfs(Digraph G, int v) {
    low[v] = dfn[v] = time++;
    marked[v] = true;
    inStack[v] = true;
    stack.push(v);
    for (int w : G.adj(v)) {
        if (!marked[w]) {
            dfs(G, w);
            low[v] = Math.min(low[v], low[w]);
        } else if (inStack[w]) {
            low[v] = Math.min(low[v], dfn[w]);
        }
    }
    if (low[v] == dfn[v]) {
        ArrayList<Integer> temp =new ArrayList<Integer>();
        int j = -1;
        while (v != j) {
            j = stack.pop();
            inStack[j] = false;
            temp.add(j);
        }
        result.add(temp);

    }
}

public List<ArrayList<Integer>> result(){
    return result;
}
